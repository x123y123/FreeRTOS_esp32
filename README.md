# ESP32 with FreeRTOS
Let's get started learning FreeRTOS with esp32~

## Install ESP-IDF on Mac m1
```shell
$ brew install cmake ninja dfu-util
# For faster builds
$ brew install ccache

# Get esp-idf repo
$ mkdir -p ~/esp
$ cd ~/esp
$ git clone -b v5.1 --recursive https://github.com/espressif/esp-idf.git

# Set up the tools

```
